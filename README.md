# Pixeled Compiler

Based on Pixeled's (youtube.com/@pixeled-yt) "Creating a Compiler" series.

- `v1` Corresponds to timestamp of 0:00-19:55 in the video series. At this point, the directory contains the .asm file of a program to return an exit code, object file output by NASM (`nasm`), executable produced by GNU Linker (`ld`), and a Bash script to automatically check the program.
